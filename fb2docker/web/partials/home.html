B2Docker Fedora Remix

A fast moving, minimized operating system acting as a Docker host.

Features

  • Tracking latest Docker builds and testing
  • Built using Fedora RPM technology on the server side for a known base, but with OSTree on the client side for fully atomic upgrades.
  • A familiar Unix environment, with a writable etc
  • The ability to choose between multiple variant trees, with additional RPM content.
