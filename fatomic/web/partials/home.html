Fedora Atomic Initiative

The goal of this initiative is to accelerate development and increase reliability of a subset of the packages in Fedora, using OSTree, and more specifically the rpm-ostree integration project.

OSTree is a new basis for operating system deployment - it offers fully atomic upgrades, and reliable replication of known state. Using rpm-ostree, (mostly) unmodified existing packages can be placed into an OSTree repository.
